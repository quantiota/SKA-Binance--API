# SKA C++ Engine

Development folder for the native C/C++ implementation of the SKA Engine.

This workspace is dedicated to building a low-latency version of the engine for real-time processing of raw tick data, entropy computation, structural probability analysis, regime transitions, and trading logic integration.

## Purpose

* implement the SKA Engine in C/C++
* reduce latency and improve execution speed
* control memory and stream processing directly
* prepare a native core for real-time market applications

## Scope

The engine is intended to support:

* raw tick data ingestion
* entropy computation
* structural probability computation
* regime classification
* transition detection
* trading bot integration
